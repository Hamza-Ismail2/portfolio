SyncIt was a Spring 2025 group project focused on building a private, local-network cloud alternative for homes and offices. The core idea was simple: set up one machine as a server, and allow other devices on the same network to connect, upload, and access shared data.

The project was designed for practical day-to-day use, especially when mobile devices run out of storage. Instead of relying only on third-party cloud services, users could keep a computer at home or in the office as their own storage hub.

Key highlights:

- one device configured as a central server on the local network
- peer device access for file sharing and retrieval
- private, in-network storage for home and office environments
- useful for extending storage for low-capacity mobile devices
- multiple testing scenarios completed to validate reliability and behavior

This project helped me strengthen teamwork, networking fundamentals, and system testing practices in a real multi-device setup.
