![Chrome Dino DQN agent gameplay](/chrome-dino-dqn.png)

This AI project focused on reinforcement learning, where we built a Deep Q-Network (DQN) model that learned to play the Chrome offline dinosaur game autonomously.

Instead of hardcoded rules, the agent improved through trial and error by interacting with the environment, collecting rewards, and updating Q-values over many training episodes. Over time, it learned better jump decisions and obstacle handling strategies.

Key highlights:

- DQN-based game-playing agent for the Chrome Dino environment
- epsilon-greedy exploration strategy for balancing exploration and exploitation
- replay memory and iterative training to stabilize learning
- performance improvement observed across training runs
- collaborative AI project implementation and testing

This project gave me practical experience with reinforcement learning workflows, reward-driven optimization, and evaluating agent behavior in a real interactive task.
