This project was built in Fall 2025 (November) to learn the Java Spring Boot framework through a practical, backend-focused application.

I developed a CLI-based expense tracker inspired by Splitwise, where users can manage shared expenses, track who owes whom, and maintain clear settlement balances across participants.

Key highlights:

- Splitwise-like shared expense and balance tracking features
- Spring Boot architecture with clean service and API layers
- CLI-driven interaction for adding expenses and checking settlements
- Swagger UI enabled for exploring and testing exposed REST endpoints
- built as a learning project to strengthen Java + Spring Boot fundamentals

Since this was a command-line project, there is no UI screenshot. The main focus was backend design, business logic correctness, and API usability.
