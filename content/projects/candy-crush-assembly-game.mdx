This project was developed for my **Computer Organization and Assembly Language (COAL)** coursework in Fall 2023.
I designed and implemented a **Candy Crush style game entirely in Assembly language**, which was one of the most challenging systems I built at that stage.

The game uses a terminal-based visual interface with a color grid layout to simulate interactive gameplay:

- **CLI Color Grid UI:** built a text-based, color-rendered board to represent candies and game state.
- **Match Detection Logic:** implemented checks for horizontal and vertical matches.
- **Swap + Validation Rules:** handled user moves, legal swaps, and invalid move rejection.
- **Scoring System:** updated score dynamically based on successful pattern clears.
- **Board Update Cycle:** cleared matched items and refreshed the grid for continued play.
- **Game Flow Handling:** managed input loops, progression, and replay-style behavior.

Building this in Assembly strengthened my understanding of **low-level memory handling, control flow design, state management, and structured problem-solving under tight language constraints**.
